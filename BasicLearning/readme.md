### 对象和类
状态不同但是其它方面相似的，如账户是一个类，都有金额这一项，但是不同对象金额的数量不同(状态不同)
类描述了具有相同特性和行为(功能)的对象集合，所以一个类实际上就是一个数据类型，例如所有的浮点型数字具有相同的特性和行为集合。
